# Личный кабинет студента

## Описание проекта
Веб-приложение для управления учебным процессом студентов и преподавателей.

## Основные цели
- Централизация учебных материалов и расписания
- Автоматизация процесса оценивания
- Упрощение коммуникации между студентами и преподавателями

## Функционал
### Для студентов:
- Просмотр расписания занятий
- Отслеживание успеваемости и оценок
- Загрузка выполненных заданий
- Доступ к учебным материалам

### Для преподавателей:
- Выставление оценок
- Публикация заданий и материалов
- Формирование отчётов
- Управление учебными курсами

## Технологический стек
- Backend: Java Spring Boot
- Frontend: Thymeleaf + Bootstrap
- База данных: PostgreSQL
- Сборка: Maven
